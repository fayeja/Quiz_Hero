1 What's the difference between .forEach(), .map(), .filter(), .find() ?
  • .forEach() executes code on each element of an array but does not modify the array and returns
    undefined.
  • .map() runs the same code on each element of an array and returns a new array with the modified elements.
  • The function .filter() checks each element in an array to determine if it matches a set of criteria and returns
    a new array with the elements that match.
  • The function.find() checks each element in an array to see whether it matches a set of criteria and returns
    the first matching element.

    2 What's the difference between local storage,section storage ,cookies?
  • localStorage is a method of storing data on the computer of the client. It allows key/value pairs to be
    saved in a web browser and stores data with no expiration date. JavaScript and HTML5 are the only ways
    to access localStorage. However, the user can clear the browser data/cache to delete all localStorage data.
  • Section storage only stores data for a single session, which means it is only saved until the browser (or tab)
    is closed. Data is never sent to the server and can only be read on the client side.
  • Cookies are generally for server-side reading (although can also be read on the client site), whereas
    localStorage and sessionStorage are usually read on the client. Data that must be given back to the server with
    subsequent XHR requests is saved here. Its expiration time varies depending on the kind, and the length can be
    defined either server-side or client-side.
